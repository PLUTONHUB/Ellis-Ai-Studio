# Universal Content Framework

## Source → Story → Format → Review

1. Select a verified source event.
2. Extract the audience problem, outcome, and approved proof.
3. Choose a shared story, hook, CTA, and hashtag set.
4. Adapt the message through the relevant platform template or prompt.
5. Review facts, voice, accessibility, visual permissions, and platform-specific rules.
6. Publish and retain the final asset and learnings as an example.

The source event remains the factual record. The shared engine keeps the message consistent. The platform module changes only the format and distribution approach.
