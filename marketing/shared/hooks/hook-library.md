# Hook Library

- “We built this because [customer problem].”
- “The growth opportunity in [workflow] is [opportunity].”
- “Here’s what changed when we made [outcome] easier.”
- “Before you [common action], consider this.”
- “A better [customer journey] starts with [insight].”
- “What looks like a small change can create a meaningful business outcome.”

Adapt the hook to the source material. The first line must be supported by the post.
