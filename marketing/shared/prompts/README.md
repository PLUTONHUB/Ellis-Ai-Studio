# Shared Prompt Inputs

Use these inputs with every platform generator:

- Source: commit, pull request, project update, or service launch
- Audience and business context
- Chosen message from `brand/messaging.md`
- Chosen story and hook
- Approved evidence, links, visual assets, and claims
- Platform, format, and desired call to action

Always instruct the generator not to invent facts, metrics, customer names, availability, or timelines.
