# Hook Intelligence Library

`marketing/shared/hooks/` remains the canonical hook library. This folder organizes evidence and experiments by hook type:

- Curiosity
- Story
- Contrarian
- Educational
- Authority
- Pain Point
- Case Study

For each result, link the shared hook or content ID, platform, audience, format, metric, comparison set, and recommendation. Promote only validated learnings back to the shared library through review.
