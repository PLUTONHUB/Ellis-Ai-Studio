# Marketing Intelligence Layer

The Intelligence Layer turns performance data, research, and ideas into evidence-backed recommendations. `marketing/shared/` remains the single source of truth for brand, messaging, positioning, hooks, storytelling, CTAs, and hashtags; intelligence observes and recommends rather than redefining it.

## How intelligence feeds operations

- **Campaigns:** Prioritize audiences, messages, formats, and experiments using validated opportunities.
- **Content Library:** Tag content with source, outcome, evergreen/seasonal status, and reuse potential.
- **Scheduling:** Recommend cadence and posting windows only after enough comparable data exists.
- **Repurpose Engine:** Identify high-performing and evergreen source material worth adapting.
- **Automation:** Supply approved trigger context, recommendation rules, and review requirements; never bypass human approval.

## Subsystem integration map

| Subsystem | Primary operational destination |
| --- | --- |
| Content intelligence | Content library, scheduling, repurpose, campaigns |
| Audience | Shared-engine review, campaigns, content briefs |
| Competitors | Campaign opportunity analysis and experiments |
| SEO | Website, content library, campaigns, and calendar |
| Trends | Calendar, campaigns, repurpose, and experiments |
| Research | Campaign briefs, content drafts, and shared-engine review |
| Keyword library | SEO work, website opportunities, and content briefs |
| Hook library | Shared hooks, experiments, and platform generators |
| Idea vault | Content library drafts, campaigns, and automation candidates |
| Experiments | Scheduling, campaigns, analytics, and shared-engine review |
| Insights | Campaigns, content library, scheduling, repurpose, and automation |
| Reports | Operating reviews and future planning across every subsystem |

## Operating loop

```text
Collect → Organize → Analyze → Recommend → Test → Learn → Reuse
```

Each finding should cite its source, date range, confidence, affected channel or audience, recommended action, owner, and follow-up measurement.
