# Instagram Content Workflow

This folder converts meaningful product work into reusable Instagram content. Templates establish the content shape, prompts generate a first draft, and the format folders hold review-ready assets.

## Shared foundation

Before drafting, use `../shared/` for Ellis AI Studio brand voice, messaging, storytelling, hooks, CTAs, hashtags, and approved assets. This module adapts that source of truth for Instagram formats rather than duplicating it.

## Folder guide

- `templates/` — Reusable structures for campaign and format types.
- `prompts/` — Copy-and-use prompts for generating content from GitHub activity and portfolio updates.
- `captions/` — Long and short caption drafts.
- `reels/` — Reel scripts, shot lists, and voiceover drafts.
- `carousel/` — Carousel slide outlines and final copy.
- `stories/` — Story sequences and interactive sticker copy.
- `hashtags/` — Approved hashtag sets and performance notes.
- `examples/` — High-quality examples and annotated references.

## Workflow

```text
GitHub Commit
      ↓
Instagram Draft
      ↓
Review
      ↓
Publish
```

1. Select a meaningful commit, merged pull request, or portfolio update with a clear audience benefit.
2. Choose the closest template and generation prompt, then save the output in its format folder with a descriptive date-based filename.
3. Review accuracy, visual direction, brand voice, calls to action, accessibility, links, and hashtags.
4. Publish using the approved Instagram workflow. Preserve final copy, scripts, and learnings as examples for future reuse.

Do not publish secrets, internal-only context, unverified metrics, or customer details without approval.
