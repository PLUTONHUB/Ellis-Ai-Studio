# Feature Release → Reel Script

```text
Turn this feature release into a 20–30 second Instagram Reel.

Audience: [target audience]
Feature release details: [paste]
Approved outcomes, metrics, and links: [paste]

Return:
1. A time-coded reel script with a 0–3 second hook.
2. Visual direction for each beat.
3. On-screen text and optional voiceover.
4. A short caption, CTA, and 3–8 relevant hashtags.
5. Three story frames that promote the reel.

Rules: Make the benefit understandable without sound; use no unverified claims; exclude secrets and internal-only context.
```
