# GitHub Commit → Instagram Caption

```text
Create Instagram content from this GitHub commit.

Audience: [target audience]
Content type: [Project Launch | Before & After | Behind the Build | Feature Highlight]
Commit message: [paste]
Diff or release summary: [paste]
Approved context, metrics, and links: [paste]

Return:
1. A long caption (120–180 words).
2. A short caption (40–70 words).
3. One clear call to action.
4. 3–8 relevant hashtags.

Rules: Lead with the audience benefit; use plain language; do not invent facts, metrics, names, or timelines; exclude secrets and internal-only context.
```
