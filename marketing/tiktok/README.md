# TikTok Content Workflow

This module turns meaningful product and client work into short-form video content. Use templates to choose a story, prompts to generate a first draft, and the asset folders to prepare recording-ready material.

## Folder guide

- `templates/` — Reusable story structures.
- `scripts/` — Timed scripts, shot lists, B-roll, on-screen text, and thumbnail ideas.
- `hooks/` — Opening-line variations.
- `captions/` — Caption and call-to-action blocks.
- `hashtags/` — Approved hashtag sets.
- `trends/` — Trend research and adaptation notes.
- `prompts/` — Generation prompts for engineering, portfolio, feature, website, and client updates.
- `examples/` — Approved examples and performance learnings.

## Workflow

```text
GitHub Commit
      ↓
TikTok Script
      ↓
Review
      ↓
Record
      ↓
Publish
```

1. Pick a meaningful commit, pull request, project update, or launch with a clear viewer benefit.
2. Choose a story template and matching prompt, then save generated assets in their format folder using a descriptive date-based filename.
3. Review accuracy, hook, visual claims, voice, accessibility, music/trend usage, captions, CTAs, and hashtags.
4. Record and publish through the approved TikTok workflow. Preserve final assets and results as examples.

Never expose secrets, private client details, unapproved metrics, or internal-only information.
