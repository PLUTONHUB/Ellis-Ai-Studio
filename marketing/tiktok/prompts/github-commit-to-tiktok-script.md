# GitHub Commit → TikTok Script

```text
Turn this GitHub commit into TikTok content.

Audience: [target audience]
Story type: [choose a template]
Commit message: [paste]
Diff or release summary: [paste]
Approved facts, metrics, and links: [paste]

Return a 15-second, 30-second, and 60-second script; five hook variations; on-screen text; B-roll suggestions; a shot list; one caption; one CTA; 3–6 relevant hashtags; and three thumbnail title ideas.

Rules: Lead with viewer value, not implementation details. Do not invent facts, metrics, names, or timelines. Exclude secrets and internal-only context.
```
