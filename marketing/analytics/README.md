# Analytics Framework

Analytics measures whether messages from `marketing/shared/` and campaign choices create useful audience outcomes. Keep source data and reporting notes here; do not rewrite brand guidance based on one data point.

- `platform-metrics/` — Channel-level reach, views, followers, and post results.
- `campaign-performance/` — Campaign summaries against their defined goals.
- `engagement/` — Saves, shares, comments, click-through signals, and best-performing hooks.
- `conversions/` — Website traffic, leads, qualified inquiries, and conversion events.

## Review framework

For each reporting period, record the source, date range, audience, content IDs, campaign IDs, engagement, traffic, leads, conversions, best hooks, CTA performance, and next test. Compare like with like and label unavailable data rather than estimating it.
