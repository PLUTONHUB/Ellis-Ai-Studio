# Repurpose Engine

Repurposing adapts one verified source event into several formats while preserving the shared message and approved facts. Start with `marketing/shared/`, then create platform-specific drafts without copying unreviewed content between channels.

## Core workflow

```text
GitHub Commit
↓
Long-form Content
↓
LinkedIn
Instagram Carousel
Instagram Reel
TikTok
Google Business Profile
Website Changelog
Portfolio Update
Newsletter
Blog Article
```

- `workflows/` — Repeatable source-to-format paths.
- `templates/` — Reusable repurposing maps and checklists.
- `examples/` — Approved source-to-output examples and lessons.

Validate the facts, CTA, audience, and platform rules at every adaptation step.
