# Design Prompt Library

Each template uses `{campaignTitle}`, `{campaignSummary}`, `{audience}`, `{cta}`, `{platform}`, and `{format}`. Append the canonical [Brand Design System](BRAND_DESIGN_SYSTEM.md) rules to every prompt.

| Template | Parameterized prompt intent |
| --- | --- |
| Instagram / Facebook / X / Threads | `{format} promotional graphic for {campaignTitle}; communicate {campaignSummary}; audience {audience}; CTA {cta}` |
| LinkedIn | `{format} executive carousel concept for {campaignTitle}; evidence-led business value: {campaignSummary}` |
| TikTok cover / YouTube thumbnail | `{format} high-clarity cover for {campaignTitle}; one focused business outcome; CTA {cta}` |
| Ads / Stories | `{format} conversion-first campaign visual for {campaignTitle}; audience {audience}; CTA {cta}` |
| Carousel | `{format} 5-slide narrative: problem, insight, system, proof, CTA for {campaignTitle}` |
| Portfolio / Case study | `{format} premium before/after or outcome showcase for {campaignTitle}; document {campaignSummary}` |
| Service promotion | `{format} service-value graphic for {campaignTitle}; outcome and next step {cta}` |
