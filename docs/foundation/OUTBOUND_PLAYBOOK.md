# Outbound Playbook

Outbound targets qualified businesses using researched, permission-aware, relevant messages. Each sequence has a defined audience, hypothesis, value proposition, channel, cadence, opt-out process, and measurement plan. Outreach records and consent are owned by Sales CRM/Outreach; do not maintain shadow lists.

Use [Brand Positioning](BRAND_POSITIONING.md), [Ideal Client Profile](IDEAL_CLIENT_PROFILE.md), and [Sales Playbook](SALES_PLAYBOOK.md) as authoritative inputs.
