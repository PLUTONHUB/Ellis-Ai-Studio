# Core Values

1. **Measured outcomes over activity:** optimize for business impact with visible evidence.
2. **Systems over one-offs:** create reusable capabilities and document justified exceptions.
3. **Truthful partnership:** set clear expectations, protect client trust, and communicate constraints.
4. **Craft with leverage:** combine thoughtful human judgment with responsible automation.
5. **Continuous learning:** preserve evidence, learn from results, and improve the system.

These values govern delivery in [Client Delivery Standard](CLIENT_DELIVERY_STANDARD.md) and AI operation in [AI Worker Contracts](ai-worker-contracts.md).
