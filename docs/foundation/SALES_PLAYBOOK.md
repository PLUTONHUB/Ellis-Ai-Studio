# Sales Playbook

1. Qualify against the [Ideal Client Profile](IDEAL_CLIENT_PROFILE.md).
2. Diagnose the measurable constraint, current process, decision process, urgency, and capacity to act.
3. Recommend the smallest reusable system that can prove value.
4. Run [Discovery](DISCOVERY_CALL_PLAYBOOK.md), issue a compliant [Proposal](PROPOSAL_STANDARD.md), and record the outcome in Sales CRM.
5. On acceptance, hand off only the canonical discovery record to [Client Onboarding](CLIENT_ONBOARDING.md).

Never promise outcomes without agreed evidence, dependencies, and client responsibilities.
